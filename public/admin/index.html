Масова розсилка від імені школи

Вставте адреси (з Excel — просто скопіюйте стовпець), напишіть повідомлення та натисніть «Надіслати». Листи йдуть по одному з паузою ~1,5–2 с — кожен отримувач бачить лише свою адресу, тому лист не виглядає як спам. Адреси ніде не зберігаються — використовуються один раз лише для цієї розсилки. Орієнтовний ліміт Gmail — ~400–500 листів на добу.

Тексти листів, які сайт надсилає автоматично. Підстановки на кшталт {{childName}} замінюються реальними даними (клікніть по підстановці, щоб вставити її в текст). Порожня тема = лист вимкнено.

Редагування

Натисніть на будь-який текст на полотні, щоб редагувати його прямо там — заголовки, описи, розсилку, футер тощо.

Натисніть на секцію з фото чи картками (Головна, Про нас, Курси, Викладачі, Галерея, Відгуки, FAQ) — її поля відкриються тут.

Головний банер: фото, фонове зображення, заголовок і показники. Виділіть слово зірочками, напр. *майбутнє*.

Секція «Про нас»: фото, заголовок, текст та переваги.
